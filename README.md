# 互联网+智慧物流质询系统

管理物流系统的方案、节点、道路和物品，根据最短路径和优先级智能发货。

[立刻访问](https://intelligent-logistics.mrcai.dev)

## ✨ 项目介绍

1. 最短路径计算（Dijkstra）
2. 数据库并发（信号量、强两段锁协议）
3. SQL 解析器（递归下降文法）
4. 前后端（Express + React）
